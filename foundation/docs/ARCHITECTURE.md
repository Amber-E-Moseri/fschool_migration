# Foundation School System Architecture

## Overview
The Foundation School platform is a multi-campus discipleship and education management system.

### Current Stack
- Google Apps Script backend
- Google Sheets datastore
- Static HTML/CSS/JS portals
- Mailchimp integration
- Moodle integration planning
- Planned Supabase migration

## Core Modules
1. Registration System
2. Teacher Availability Scheduler
3. Attendance System
4. Reporting System
